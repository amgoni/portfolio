FEATURED PROJECTS

Studox

Studox is a web application that allows students to search for and access study materials for their courses. It was developed using React and Firebase.
Studox offers a simple and intuitive interface for students to search for study materials, access detailed information about them, and easily navigate through search results.

Jourknow

Jourknow is a user-friendly web tool designed for searching and exploring research journals across different fields of study. Developed using React, Jourknow seamlessly connects to the CORE journal repository through the CORE API.
It offers a straightforward interface for users to search, access detailed information about journals, and easily navigate through search results.

Task Tracker

Task Tracker is a simple web application that allows users to add, delete, and toggle reminders for tasks. It was developed using React and Firebase's realtime database.
Task Tracker offers a straightforward interface for users to add tasks, set reminders, and toggle reminders on and off.
